# 🧩 Task Management API

API REST para la gestión de proyectos y tareas.

---

## 🚀 Tecnologías

* Node.js
* Supabase
* JWT
* dotenv

---

## 📦 Instalación

```bash
git clone https://github.com/CristhianJonathanDelCastilloAlba/task-management-back.git
cd task-management-back
npm install
```

---

## ⚙️ Configuración

Crear archivo `.env`:

```env
PORT=3000
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=123456
DB_NAME=task_management
JWT_SECRET=secret
```

---

## ▶️ Ejecutar

```bash
npm run dev
```

## 🔐 Autenticación

Usa JWT:

```
Authorization: Bearer <token>
```
